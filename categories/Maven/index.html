3rd Party library 의 dependency 추가 (local repository 사용)

Mar 08, 2018 in Maven

3rd Party library 사용

maven repository 에서 관리되지 않는 라이브러리를 사용하게 되었다. (Altibase jdbc driver)
라이브러리 관리를 위해 maven 의 pom.xml 에 dependency 를 추가하고 싶을 때가 있다. 여러 방법이 있지만, 아래 방법으로 해결하였다. (형상 관리 측면에서도 괜찮다고 생각한다.)

설정 방법

아래 방법으로 설정한다.

pom.xml 수정

maven 의 pom.xml 에 아래 내요을 추가한다.

디렉토리 생성 및 라이브러리 복사

프로젝트 내에, src\main\repository\Altibase\jdbc\driver\altibase\5.1.5.46 경로의 디렉토리를 생성한다.

  • src\main\repository 는 repository 의 base 경로이다.
  • Altibase\jdbc\driver 라이브러리의 패키지명이다.
  • altibase\5.1.5.46 라이브러리 파일명이다.

라이브러리를 생성한 디렉토리에 복사한다.

  • altibase-5.1.5.46.jar

메이븐 빌드

위와 같이 설정을 하면, pom.xml 에 에러가 발생하는 경우가 있다. 이를 해결하기 위해서 maven goal(mvn package) 를 실행한다.

확인

Maven Dependencies 에 라이브러리가 복사되어 있는지 확인한다.
